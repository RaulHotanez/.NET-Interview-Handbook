### Tipos de dato primitivos
Los tipos primitivos en C# son los tipos básicos que proporciona el lenguaje para almacenar datos simples. En una entrevista normalmente esperan que puedas nombrarlos y explicar para qué sirve cada uno.

```csharp title="C#"
Tipo	Alias	Tamaño	Ejemplo
bool	Boolean	1 byte (conceptualmente)	true, false
byte	Byte	1 byte	255
sbyte	SByte	1 byte	-100
short	Int16	2 bytes	32000
ushort	UInt16	2 bytes	65000
int	Int32	4 bytes	100
uint	UInt32	4 bytes	100U
long	Int64	8 bytes	100000L
ulong	UInt64	8 bytes	100000UL
float	Single	4 bytes	3.14f
double	Double	8 bytes	3.14159
decimal	Decimal	16 bytes	99.99m
char	Char	2 bytes	'A'

```
